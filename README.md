<h1>getNQA</h1>
<p>getNQA est un outil qui permet de recuperer et de formaliser les donnees NQA des routeurs H3C.</p>
<p>Cet outil a ete realise en C# dans l'objectif d'une Metrologie des Reseaux.</p>
<p></p>
<h2>Utilisation</h2>
<pre>getNQA -h</pre>
<p></p>
<h2>Version</h2>
<pre>getNQA --version</pre>
